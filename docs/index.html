Minesweeper: GitHub, JavaFX, Rekursion

3. Klasse
Durch einen Linksklick der Maus auf ein Feld, öffnen sich alle Felder rundherum die leer sind und das jeweilige, wenn keine Bombe dort ist.
Mit einem Rechtsklick der Maus kann ein Feld mit einer Bombe markiert werden.
Wenn man auf ein Feld mit einer Bombe einen Linksklick macht, verliert man und alle Bomben zeigen sich.

Rekursion


Rekursive Ansatz mittels Methode: 

						public class Rekursiv {

    						public static int i = 0;

    						public static void rekursiveMethode() {
        						i++;
        						System.out.print(i + " ");
        						if (i < 10) {
            					rekursiveMethode(); // Rekursion!
        						}
    						}

   							public static void main(String[] args) {
        						rekursiveMethode();
    						}

						}

Snake: SCRUM, JavaFX

4. Klasse
Bei Start des Spiels erscheint ein Menü bei dem man zum Spiel, den Einstellungen oder das Spiel verlassen kann.
Wenn man auf das Einstellungsrad klickt, kommt man zu den Einstellungen, bei denen man die Music ein- und ausschalten und die Lautstärke der Musik regeln kann.
Wenn man auf Play klickt, erscheint die Spielfläche. Nun kann man mit den Pfeiltasten die Schlange steuern.
Wenn man mit der Schlange einen Apfel frisst, wächst diese, der Score geht hoch und ein neuer Apfel erscheint.
Sollte man den Spielrand oder den Schlangenkörper mit dem Kopf der Schlange berühren, ist das Spiel vorbei und ein Game Over Bildschirm erscheint.

SCRUM

Scrum ist eine Methode zur Entwicklung von Projekten, bei der Teams in kurzen Zeiträumen arbeiten, um regelmäßig Fortschritte zu machen und sich flexibel an Veränderungen anzupassen.
